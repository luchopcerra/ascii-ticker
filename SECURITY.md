# Security Policy

## Reporting a Vulnerability

Please report security issues privately by emailing the maintainer or opening a private security advisory on GitHub if available.

Do not open a public issue for suspected vulnerabilities.

Include:

- A description of the issue.
- Steps to reproduce it.
- The affected route, query parameter, or configuration.
- Any known impact.

The maintainer will acknowledge the report as soon as practical and coordinate a fix before public disclosure.

## Scope

This policy covers the Worker code in this repository and project configuration such as `wrangler.jsonc`.
